React Native Mobile App

Product Explorer App

A mobile app built with React Native CLI, TypeScript, Redux Toolkit, and AsyncStorage for browsing products, searching, and saving favorites.

View GitHub Repository Open README

What This Link Shows

This Render deployment hosts a showcase page for the mobile app. The app itself runs on Android and iOS devices or emulators, not in the browser.

Features

  • Paginated product catalog from a public API
  • Responsive product search
  • Favorites with local persistence
  • Lifecycle-aware data restore
  • Redux Toolkit state management

Tech Stack

  • React Native CLI
  • TypeScript
  • Redux Toolkit + React Redux
  • React Navigation
  • AsyncStorage
